Getting Started With GitHub

What is GitHub?

GitHub is a web site that allows users to store and share files, applications and projects. The technology behind GitHub can be daunting, but for our class we will attempt to keep things simple.

We will use GitHub as a distribution point for our websites as we build them. GitHub will not be our hosting as is the case in some previous classes.

If you already have a GitHub account, do not create another. Instead, create a repository to be used for our class.

The Video

Watch the video below and follow along with it. When done, you should have a GitHub account and repository for use in the class.
