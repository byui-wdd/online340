pgAdmin 4

Introduction

While render.com provides the server to host our database, it doesn't provide any tools to interact with the database. We will use pgAdmin 4 as our primary development tool. This is a free software application specifically designed for PostgreSQL databases.

Download pgAdmin 4

  1. Navigate in a browser tab to pgAdmin.org.
  2. Click the "Download" option on the left.
  3. Click the option for your operating system from the top row of options.
  4. Click the newest version (top item) from the list.
  5. In the new window, click the bottom item under the Files heading. For Mac it should have a "dmg" extension, for Windows it should have an "exe" extension.
  6. Wait for the download to finish.

Install

  1. Open the downloads folder on your computer.
  2. Find and double-click the pgAdmin installer.
  3. Follow the default installer options for your operating system.
  4. Important! During the installation you will be asked to supply a "master password". I suggest something simple, like "password" for now. Store this password somewhere so you can find it in the future if needed. If you forget it, it is very difficult to fix.

Conclusion

With pgAdmin installed, we'll start using it in the next activity.
