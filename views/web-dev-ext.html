FireFox Installation

For FireFox users, follow these steps to install the extension:

  1. You can locate it by going to FireFox > Menu > Add-ons > Extensions
  2. Type "Web Developer" in the search box (top right corner)
  3. Install the one from Chris Pederick.
  4. Or, use https://addons.mozilla.org/en-US/firefox/addon/web-developer/
  5. It works exactly the same as the Chrome extension shown in the video.
