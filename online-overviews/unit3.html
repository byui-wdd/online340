Unit 3 Overview

Using Express and SQL with MVC

This unit expands and applies the concepts from the previous weeks. You will be introduced to the Model - View - Control architecture to build web applications. You will build an Express / Database process to replace the existing static navigation bar in your template and existing views with content from the database. In addition, you will do some basic debugging and apply error handling to serve generic error messages when they occur. Finally, you will deploy the application to a live server accessible on the Internet.
