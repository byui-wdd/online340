Unit 2 - Databases & SQL

Introduction

The unit is an introduction to the PostgreSQL database as well as the basic concepts of databases: tables, columns, records, relationships, and keys using the PGAdmin tool. In addition, SQL SELECT, INSERT, UPDATE, and DELETE statements will be introduced, as will CREATE, ALTER and DROP.
